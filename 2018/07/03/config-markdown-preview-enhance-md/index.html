配置Markdown Preview Enhanced

配置Markdown Preview Enhance是功能最强大的Atom markdown插件，提供了多个的markdown解析以及导出功能。对于强迫症患者并不希望不能用（功能不完善的）插件在眼前晃。测试多个markdown导出工具后最终选择，使用PhantomJS（和Mermaid兼容性最好）最为主力导出工具。

屏蔽Markdown Preview Enhanced预览菜单

Markdown Preview Enhance支持多种解析、导出工具，及时不使用也无法在预览界面中关闭，但是可通过修改Markdown Preview Enhanced插件实现，具体操作方法如下：

  1. 在Atom的插件目录中查找webview.js文件
  2. 查找 initContextMenu()函数
  3. 注释不使用的导出工具菜单
